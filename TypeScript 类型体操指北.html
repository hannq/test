基本运算

集合运算

A \(\supset\) B

使用 extends 关键字
例：

type Test<T> = T extends boolean ? 'Y' : 'N';

Test<boolean>; // Y
Test<string>; // N
A \(\cap\) B

使用 & 符号
例：


type Test = { a: string } & { b: number }

// ==> 等价于

type Test = {
    a: string;
    b: number;
}

常见运用：用于合并一个或多个类型

A \(\cup\) B

使用 | 符号
例：

type Test = { a: string } | { b: number }

常见运用：用于描述某些不确定类型

A \(\cap\) \(\emptyset\) = \(\emptyset\)

例：

type Test = { a: string } & never;

// ==> 等价于

type Test = never;

常见运用：用于剔除某些不需要的类型

A \(\cup\) \(\emptyset\) = A

例：

type Test = { a: string } | never;

// ==> 等价于

type Test = { a: string }

常见运用：用于剔除某些不需要的类型，比如某些「内置范型操作符」：Exclude、Extract、Omit 等

在常见的编程语言中，我们常常需要对 Array、Tuple 结构进行 循环/遍历，语言本身也内置了很多方法帮助我们完成这样的需求。但是在 TypeScript 类型系统 中，并不支持这样直接地 循环/遍历 像 Array、Tuple 这样的结构。

递归运算

自引用递归

通过简单的词法表达即可完成自引用递归
例：

interface Cate {
    id: string;
    name: string;
    children: Cate[];
}

常见运用：用于描述无限层级嵌套的结构，如：一级分类、二级分类

尾递归

可以利用「泛型操作符」即可实现简单的尾递归

例：使用尾递归实现 [A, B] => A & B 的转化

// shift action
type ShiftAction<T extends any[]> = ((...args: T) => any) extends ((arg1: any, ...rest: infer R) => any) ? R : never;

type combineTupleTypeWithTecursion<T extends any[], E = {}> = {
  1: E,
  0: combineTupleTypeWithTecursion<ShiftAction<T>, E & T[0]>
}[T extends [] ? 1 : 0]

type test = [{ a: string }, { b: number }];
type testResult = combineTupleTypeWithTecursion<test>; // { a: string; } & { b: number; }

常见运用：用于操作不定个数的类型「叠加操作」

利用 infer 关键字进行复合运算

infer 用于推导指定位置的类型
简单的来说，你可以把 Promise<inter A> 粗略理解为 var A = Promise[Symbol.Generic]1

常用技巧

常见提取技巧

快速取出 Promise 泛型
type DemoType = Promise<string>;

// 比较啰嗦的做法，而且不是那么通用
type ObtainPromiseResolveType<T> = T extends Promise<infer R> ? R : never;
type TT = ObtainPromiseResolveType<DemoType>; // string

// 简便做法 
// TypeScript 4.5 以后新增
type TT = Awaited<DemoType>; // string
快速取出 Array 泛型
type DemoType = Array<string>;

// 比较啰嗦的做法
type ObtainPromiseResolveType<T> = T extends Array<infer R> ? R : never;
type TT = ObtainPromiseResolveType<DemoType>; // string

// 简便做法 
type TT = DemoType[any]; // string
// 或者
type TT = DemoType[number]; // string
